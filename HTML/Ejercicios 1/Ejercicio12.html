Evolución del Lenguaje Web

Cronología de **HTML**

La siguiente tabla resume los hitos más importantes en el desarrollo del estándar:

Versión Año Características Principales
**HTML** 1.0 1991 Primera definición pública del lenguaje con 18 elementos básicos.
**HTML** 2.0 1995 Primera versión estandarizada. Introducción de formularios y tablas básicas.
**HTML** 3.2 1997 Soporte para tablas complejas, applets de Java y texto alrededor de imágenes.
**HTML** 4.01 1999 Separación de estructura y diseño (vía CSS). Gran estabilidad durante años.
**HTML5** 2014 Inclusión nativa de multimedia, etiquetas semánticas y APIs de geolocalización.
