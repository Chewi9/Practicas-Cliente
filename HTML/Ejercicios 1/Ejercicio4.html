HTML

Historia HTML

HTML, creado por Tim Berners-Lee a finales de los años 80.
HTML, es el lenguaje de marcado que permitió el desarrollo de la World Wide Web y la creación de páginas web interconectadas.
Desde su creación, HTML ha evolucionado a lo largo de los años, con varias versiones que han introducido nuevas características y mejoras. La versión más reciente, HTML5, ha sido ampliamente adoptada y ha permitido el desarrollo de aplicaciones web más ricas y dinámicas.
En resumen, HTML ha sido fundamental para el desarrollo de la web y ha permitido a los usuarios crear y compartir contenido en línea de manera fácil y accesible.

Código HTML

Ejemplo 
            <!DOCTYPE html>
            <html>
                <head>
                    <meta charset="utf-8">
                    <meta name="author" content="JMAP">
                    <title>Ejercicio 1 HTML</title>
                </head>
                <body>
                    <h1>Historia <strong>HTML</strong></h1>
                    <p><strong>HTML</strong>, creado por <i>Tim Berners-Lee</i> a finales de los años 80.</p>
                    <p><strong>HTML</strong>, es el lenguaje de marcado que permitió el desarrollo de la World Wide Web y la creación de páginas web interconectadas.</p>
                </body>
            </html>

Características Principales de HTML

Versiones de HTML

  1. HTML 1.0 (1991)
  2. HTML 2.0 (1995)
  3. HTML 3.2 (1997)
  4. HTML 4.01 (1999)
  5. HTML5 (2014)

Ventajas de HTML5

  • Soporte para multimedia (audio y video)
  • Mejoras en la semántica y estructura del documento
  • Compatibilidad con dispositivos móviles
  • Mejoras en el rendimiento y la accesibilidad
