# 🚀 Prácticas de Frontend

Este repositorio contiene los apuntes, ejercicios y proyectos realizados durante la asignatura de **Tecnologías del Cliente para Aplicaciones Web**, centrada en la creación de interfaces modernas, accesibles y funcionales.

## 📋 Contenidos del Curso

### 1. HTML5 (Estructura y Semántica)
* **Fundamentos**: Definición de etiquetas, atributos y elementos nulos como `<img>` y `<br>`.
* **Estructura Global**: Uso correcto de `<!DOCTYPE html>`, `<html>`, `<head>` y `<body>`.
* **Semántica**: Organización de bloques con `<header>`, `<nav>`, `<main>`, `<section>` y `<footer>`.
* **Formularios**: Creación de entradas de datos con validaciones como `required`, `min/max` y tipos específicos de HTML5.

### 2. CSS3 (Estilos y Maquetación)
* **Selectores**: Uso de selectores de etiqueta, clase (`.`), ID (`#`) y selectores de atributo avanzados.
* **Modelo de Cajas**: Control total sobre `content`, `padding`, `border` y `margin`.
* **Posicionamiento**: Manejo de flujos con `static`, `relative`, `absolute`, `fixed` y `sticky`.
* **Layout Moderno**:
    * **Flexbox**: Diseño unidimensional para alineación de elementos.
    * **Grid**: Diseño bidimensional para estructuras complejas de filas y columnas.
* **Responsive Design**: Uso de `Media Queries` y unidades relativas (`rem`, `vh`, `vw`) para adaptabilidad.

### 3. JavaScript (Lógica de Cliente)
* Manipulación del DOM (Document Object Model).
* Eventos y funciones asíncronas.
* Uso de Fetch API para consumir datos externos.

### 4. React (Componentes y Estado)
* Arquitectura basada en componentes.
* Gestión de estados con `useState` y efectos con `useEffect`.
* Navegación y manejo de props.
