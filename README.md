# 🚀 Prácticas de Frontend

Este repositorio contiene los apuntes, ejercicios y proyectos realizados durante la asignatura de **Tecnologías del Cliente para Aplicaciones Web**, centrada en la creación de interfaces modernas, accesibles y funcionales.

## 📋 Contenidos del Curso

### 1. HTML5 (Estructura y Semántica)
* [cite_start]**Fundamentos**: Definición de etiquetas, atributos y elementos nulos como `<img>` y `<br>`[cite: 2, 6, 8].
* [cite_start]**Estructura Global**: Uso correcto de `<!DOCTYPE html>`, `<html>`, `<head>` y `<body>`[cite: 13, 14, 15, 22].
* [cite_start]**Semántica**: Organización de bloques con `<header>`, `<nav>`, `<main>`, `<section>` y `<footer>`[cite: 58, 59, 61, 62, 63, 64].
* [cite_start]**Formularios**: Creación de entradas de datos con validaciones como `required`, `min/max` y tipos específicos de HTML5[cite: 91, 125, 139, 140].

### 2. CSS3 (Estilos y Maquetación)
* [cite_start]**Selectores**: Uso de selectores de etiqueta, clase (`.`), ID (`#`) y selectores de atributo avanzados[cite: 319, 320, 321].
* [cite_start]**Modelo de Cajas**: Control total sobre `content`, `padding`, `border` y `margin`[cite: 240, 242].
* [cite_start]**Posicionamiento**: Manejo de flujos con `static`, `relative`, `absolute`, `fixed` y `sticky`[cite: 290, 292].
* **Layout Moderno**:
    * [cite_start]**Flexbox**: Diseño unidimensional para alineación de elementos[cite: 294, 296].
    * [cite_start]**Grid**: Diseño bidimensional para estructuras complejas de filas y columnas[cite: 302, 303].
* [cite_start]**Responsive Design**: Uso de `Media Queries` y unidades relativas (`rem`, `vh`, `vw`) para adaptabilidad[cite: 308, 310, 313].

### 3. JavaScript (Lógica de Cliente)
* Manipulación del DOM (Document Object Model).
* Eventos y funciones asíncronas.
* Uso de Fetch API para consumir datos externos.

### 4. React (Componentes y Estado)
* Arquitectura basada en componentes.
* Gestión de estados con `useState` y efectos con `useEffect`.
* Navegación y manejo de props.
